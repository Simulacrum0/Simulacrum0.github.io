What is Essence wantware?

Empowering Everyone

Wantware personally enables everyone with all computer powers of today, even A.I. "Wantware" is a computer platform for humans to have a dialog, using whatever language, special-words, and personal style of speaking, with the computer to express 'what they want their machine to do'. Its a back and forth process that establishes 'what their words mean' in a given context. Few people are programmers but all people, including programmers and technophobes, can describe in great detail what they like and dislike about a computer and more valuably, what they want to know ( why did it do that?!?!), or change ( i wish it'd just do X ) or make ( wouldn't it be cool if we could do Y ).

- Wantware uses human-described behaviors to define software
- Software uses programming code to use hardware
- Hardware uses binary code to perform tasks.
